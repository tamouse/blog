ffmpeg makes converting from Apple's proprietary M4A format to MP3 format simple:

ffmpeg -i song.m4a -ab 128000 -map_meta_data 0:0 song.mp3
raw code

Parameters:

-i file
Specifies the input file
-ab bitrate
Specifies the bitrate for the output file in bits per second. 128000 give 128kbps in the output file
-map_meta_data output:input
Writes the metadata from the input stream to the output stream. Specifying 0:0 copies the global meta data from the input file into the output file

Different on OS/X!

Of course, ffmpeg is different on OS/X. The above works on Debian-based ffmpeg.

ffmpeg -i song.m4a -ab 128000 -map_metadata:s:0 0:g song.mp3
raw code
-map_metadata
notice no underscore between meta and data!. The :s:0 means to read the global metadata and 0:g means copy it to all tracks of the output file.
