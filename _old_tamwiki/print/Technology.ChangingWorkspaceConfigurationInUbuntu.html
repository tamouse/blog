It isn't easy to configure workspaces in the stock Ubuntu 11.10 release. Starting with the article at pof.eslack.org on this very topic, a comment was given describing how to adjust workspace configuration using CompizConfig Settings Manager

1. Install CCSM

# apt-get install compizconfig-settings-manager
raw code

2. Running CCSM

$ ccsm 1>/dev/null 2>&1 &
raw code

3. Configuring Workspaces

a. Go to General Options:

b. Click on Desktop Size:

c. Select the Horizontal and Vertical Spaces

d. Click "Back"

e. Click "Close"

That should do it!
