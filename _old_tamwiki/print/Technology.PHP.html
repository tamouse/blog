PHP is a programming language for web applications that allows one to create very nice web apps in a minimal amount of fuss and bother. PHP supports many different kinds of operations, both natively and through modules that can be added to the PHP installation. PHP is one of the P's in LAMP, WAMP, XAMP, etc., servers.
